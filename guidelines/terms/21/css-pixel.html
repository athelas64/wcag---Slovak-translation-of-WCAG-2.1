CSS pixel

vizuálny uhol o veľkosti približne 0,0213 stupňov

CSS pixel je kanonická merná jednotka pre všetky dĺžky a merania v CSS. Táto jednotka je nezávislá od hustoty a odlišuje sa od skutočných hardvérových pixelov, ktoré sa nachádzajú v displeji. Nástroje používateľa a operačné systémy by mali zabezpečiť, aby bol CSS pixel nastavený čo najbližšie k referenčnému pixelu pre hodnoty a jednotky CSS modulu úrovne 3 - pozri CSS Values and Units Module Level 3 reference pixel [[!css3-values]], ktorý zohľadňuje fyzické rozmery displeja a predpokladanú vzdialenosť displeja od oka (faktory, ktoré autori obsahu nemôžu určiť).
